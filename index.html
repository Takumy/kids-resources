:: Apresentação usada na aula :: :: Estória criada pelos alunos ::
  • Scratch (scratch.mit.edu)
    Scratch é uma popular linguagem de programação visual desenvolvida pelo MIT. Permite que as crianças criem histórias interativas, animações e jogos através da manipulação de blocos de código. O Scratch é amplamente utilizado nas escolas e possui uma grande comunidade de utilizadores que partilham os seus projetos.
  • Code.org (code.org)
    O Code.org oferece uma variedade de cursos e atividades de programação para crianças de todas as idades. Possui uma secção dedicada chamada "Code.org para Educadores" que fornece recursos e ferramentas para professores e pais introduzirem a programação para as crianças. Eles oferecem cursos com personagens populares como Frozen, Minecraft e Star Wars, que podem atrair os jovens aprendizes.
  • Tynker (tynker.com)
    O Tynker é uma plataforma que oferece lições e atividades de programação para crianças. Utiliza uma interface de programação visual que ajuda as crianças a aprender conceitos de programação. O Tynker disponibiliza tutoriais interativos, jogos e projetos de programação, permitindo que as crianças criem os seus próprios jogos e animações.
  • Kodable (kodable.com)
    O Kodable é projetado especificamente para crianças mais jovens (idades entre 4 e 10 anos) e concentra-se no ensino de conceitos de programação através de jogos. As crianças aprendem a resolver quebra-cabeças e navegar por labirintos usando lógica e conceitos de programação. O site oferece uma versão gratuita com recursos limitados e uma versão premium baseada em assinatura com mais conteúdo.
  • ScratchJr (scratchjr.org)
    O ScratchJr é uma versão simplificada do Scratch, projetada para crianças pequenas. Oferece uma experiência de programação introdutória com uma interface de programação visual. O ScratchJr permite que as crianças criem histórias interativas e jogos usando personagens e cenários que adoram.
